## Asides (Notas y Advertencias)
Usa el componente Aside para resaltar información. Tiene varios tipos:

<Aside type="note" title="Nota">
Este es un Aside de tipo nota. Es perfecto para información neutral o aclaraciones importantes que el usuario no debe pasar por alto.
</Aside>

<Aside type="tip" title="Consejo Profesional">
Usa el Aside de tipo tip para dar consejos, trucos o buenas prácticas que ayuden a los usuarios a sacar el máximo provecho de tus herramientas.
</Aside>

<Aside type="caution" title="Precaución">
El tipo caution es ideal para advertir sobre acciones que pueden tener consecuencias inesperadas si no se realizan con cuidado.
</Aside>

<Aside type="danger" title="Peligro">
El Aside de tipo danger se reserva para acciones destructivas o peligrosas, como borrar datos o configuraciones críticas.
</Aside>

## Guía de Pasos (Steps)
Perfecto para tutoriales y guías paso a paso.

<Steps>
<Step title="Descargar el archivo .jar">
Obtén la última versión desde nuestra página de GitHub.
</Step>
<Step title="Instalar Dependencias">
Asegúrate de tener ProtocolLib instalado en tu servidor.
<Callout type="info">
Puedes descargar ProtocolLib desde SpigotMC.
</Callout>
</Step>
<Step title="Reiniciar el Servidor">
Un reinicio completo es necesario para que el plugin se cargue correctamente.
</Step>
</Steps>
