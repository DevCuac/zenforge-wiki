## Comandos para Jugadores

| Comando | Descripción | Alias |
| :--- | :--- | :--- |
| `/rewards` | Abre el menú principal de recompensas. | `/recompensa`, `/reward` |
| `/streak` | Abre el menú de rachas de conexión. | `/racha` |

## Comandos para Administradores

Estos comandos requieren el permiso `zenrewards.admin`.

| Comando | Descripción | Alias |
| :--- | :--- | :--- |
| `/zenrewards reload` | Recarga todos los archivos de configuración. | `/zr reload` |
| `/zenrewards reset <jugador> <id\|all>` | Reinicia el cooldown de un jugador.<br/>- **`<jugador>`**: Nombre del jugador.<br/>- **`<id\|all>`**: ID de la recompensa o `all`. | `/zr reset` |
